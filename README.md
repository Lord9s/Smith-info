# Smith Info

Smith Info is a comprehensive web information service for 2025. It provides user registration, login, and various other features.

## Features
- User Registration
- User Login
- Responsive Design

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Lord9s/Smith-info.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Smith-info
   ```
3. Open `index.html` in your browser to view the project.

## Usage
- Open `index.html` in your browser to access the main page.
- Use the Registration and Login forms to create and log into your account.
